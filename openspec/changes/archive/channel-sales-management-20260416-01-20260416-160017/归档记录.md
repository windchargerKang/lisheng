## 需求摘要

**项目名称**: 渠道销售管理系统

**目标**: 为力生服务公司打造一款管理药品生产和渠道售卖的软件系统，包括 H5 用户端和 Web 运营端。

**第一阶段范围** (本次实现):
- 用户认证模块（JWT 登录）
- 区域管理模块（树形结构）
- 店铺管理模块（分页、筛选）
- 区代管理模块（区域绑定唯一）
- 产品管理模块（三级定价）

**后续阶段**:
- H5 用户端
- 订单模块
- 分润引擎
- 新媒体主播提成模块

## 技术方案摘要

**架构**: Python FastAPI 后端 + Vue 3 双前端

**技术栈**:
- 后端：Python FastAPI + SQLAlchemy + SQLite/MySQL
- 前端：Vue 3 + Element Plus + Pinia
- 认证：JWT

**数据模型**:
- User (用户表)
- Region (区域表 - 树形结构)
- Shop (店铺表 - 含推荐关系)
- Agent (区代表 - 区域唯一绑定)
- Product (产品表)
- PriceTier (价格层级表 - 三级定价)
