# 脚本使用说明

## 数据库初始化

### 初始化数据库表结构
```bash
cd backend
PYTHONPATH=. python3 scripts/init_db.py
```

### 初始化权限数据
```bash
PYTHONPATH=. python3 scripts/init_permissions.py
```

### 初始化模拟数据
```bash
PYTHONPATH=. python3 scripts/init_mock_data.py
```

### 初始化河南省模拟数据
```bash
PYTHONPATH=. python3 scripts/init_henan_mock_data.py
```

### 创建管理员账号
```bash
PYTHONPATH=. python3 scripts/create_admin.py
```

---

## 一键初始化（全新环境）

如果是全新环境，可以按以下顺序执行：

```bash
cd backend

# 1. 初始化数据库
PYTHONPATH=. python3 scripts/init_db.py

# 2. 初始化权限数据
PYTHONPATH=. python3 scripts/init_permissions.py

# 3. 初始化模拟数据（包含管理员账号）
PYTHONPATH=. python3 scripts/init_mock_data.py
```

## 注意事项

1. **数据清理**：脚本会检查数据是否已存在，如果已存在则跳过
2. **完整重置**：需要重置数据时，先删除 `*.db` 文件再运行
3. **生产环境**：请勿在生产环境使用模拟数据脚本
4. **默认密码**：生产环境务必修改所有默认密码
