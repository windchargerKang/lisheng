## 需求摘要

将全局统一的分润比例配置改为支持按产品单独配置。在产品管理页面新增"分润配置"功能，允许运营人员为每个产品单独设置服务费比例和区代利润比例。

**验收标准**：
1. 产品管理页面新增"服务费比例"和"区代利润比例"配置项
2. 分润计算时按产品配置的比例执行
3. 产品未配置分润比例时，使用全局默认值（服务费 30%、区代利润 10%）
4. 分润比例支持小数，精确到 0.01%
5. 分润比例有合理范围限制（0% - 100%）

## 技术方案摘要

**数据模型变更**：
- `products` 表新增 `service_fee_rate` (NUMERIC(5,4)) 字段
- `products` 表新增 `agent_profit_rate` (NUMERIC(5,4)) 字段

**后端修改**：
- `Product` 模型新增分润比例字段
- `ProductCreateRequest` / `ProductUpdateRequest` / `ProductPricesRequest` 支持分润比例参数
- 产品列表/详情接口返回分润比例
- `ProfitService._calculate_profit()` 优先使用产品级配置

**前端修改**：
- 产品管理"定价"对话框新增分润配置表单
- 支持输入 0-100% 的分润比例
- 留空表示使用全局默认值

## 实现摘要

**改动文件**：
- `backend/app/models/product.py` - 新增分润字段
- `backend/app/api/v1/products.py` - API 支持分润参数
- `backend/app/services/profit_service.py` - 分润计算逻辑修改
- `frontend/src/views/Products.vue` - 定价对话框新增分润配置
